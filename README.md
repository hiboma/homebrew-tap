# hiboma/tap

Homebrew tap for tools by [@hiboma](https://github.com/hiboma).

## Usage

```sh
brew tap hiboma/tap
```

## Formulas

| Formula | Description | Version |
|---------|-------------|---------|
| [cql-lint](https://github.com/hiboma/cql-lint) | A linter for CrowdStrike LogScale (CQL) | 0.1.0 |
| [spl-lint](https://github.com/hiboma/spl-lint) | A linter for Splunk Search Processing Language (SPL / SPL2) | 0.1.0 |
| [wazuh-cli](https://github.com/hiboma/wazuh-cli) | A command-line tool for the Wazuh REST API (v4.x) | 0.1.0 |

## Install

```sh
brew install hiboma/tap/cql-lint
brew install hiboma/tap/spl-lint
brew install hiboma/tap/wazuh-cli
```
